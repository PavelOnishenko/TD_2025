Что это: минимально-играбельный TD с сеткой, одним типом башни и фиксированным путём врагов. Игрок ставит башни на свободные клетки и запускает волну. Враги идут по точкам к базе. Цель — пережить все 5 волн.

Состав:

Канвас: 720×480. Слева поле с сеткой 8×5 (тайл 50px), справа узкий сайдбар.

Путь: массив из 5–6 waypoints (полилиния); враги движутся по ней к базе.

Строительство: один тип башни (Arrow). Стоимость 10, дальность 60, урон 1, перезарядка 0.6 c.
Размещение: клик по кнопке «Place Tower (10)» → клик по свободной клетке (если хватает золота).
Подсветка только клетки под курсором: зелёная/красная (доступна/занята/недостаточно золота).

Враг: базовый юнит, постоянная скорость ~36 px/с. HP растёт по волнам.

Волны (5 шт): количество врагов — 6,7,8,9,10. Интервал спавна 0.7 c.
HP по волнам: 5, 7, 9, 12, 15.

Экономика: стартовое золото 15; за убийство +1; бонус за завершённую волну +3.

UI (минимум): верхняя панель с Lives (10), Gold, Wave X/5 и кнопкой «Next Wave». Кнопка неактивна, пока волна идёт.

Победа/поражение: при достижении базы враг отнимает 1 жизнь. Победа — после 5-й волны (и очистки поля). Поражение — при Lives = 0. Отображаем короткий текст «WIN»/«LOSE» в верхней панели и кнопку «Restart».

Рендер: простой фон, сетка линиями, путь тонкой полосой/линией, башни — кружки, враги — кружки с короткой полоской HP, снаряды — маленькие кружки. Без ассетов и эффектов.

Технологии: один index.html, Canvas 2D, без библиотек.

Игровой цикл: выбрать «Place Tower» → поставить башню → «Next Wave» → смотреть бой → тратить золото на новые башни → дожить до волны 5.
