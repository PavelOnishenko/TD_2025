# Глоссарий игровых терминов / Game Glossary

Этот документ содержит основные термины, используемые при постановке задач для игры.

---

## Основные термины / Main Terms

### Арена / Arena
**Русский:** Арена
**English:** Arena

Видимая часть экрана, в которой находится игрок, включающая "дорогу" внизу и "стену" вверху. Чёрные поля по краям экрана **не считаются** частью арены.

---

### ГГ / MC
**Русский:** ГГ (Главный герой)
**English:** MC (Main Character)

Персонаж игрока.

---

### Враги / Enemies
**Русский:** Враги
**English:** Enemies

Противники, с которыми сражается игрок.

---

### Стан / Stun
**Русский:** Стан
**English:** Stun

Анимация получения урона персонажем или врагом.
